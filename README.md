# MakerWorld Professional Keychain Generator

A fully parametric OpenSCAD keychain generator designed for MakerWorld.

## Features

- Automatic text fitting
- Smart keyring hole placement
- Multiple shapes
- Decorative borders
- Surface patterns
- Icons
- Emboss / engrave
- Print optimisation
- MakerWorld Customizer support

## Shapes

- Rounded Rectangle
- Pill
- Circle
- Heart
- Hexagon
- Bone
- Shield
- Dog Tag

## Planned

- QR Codes
- SVG Logos
- AMS Colour Inlays
- NFC Tags
- Batch Generation

## Requirements

OpenSCAD 2021.01 or newer.

## Project Structure

generator.scad is the entry point.

Everything else lives inside `/src`.
